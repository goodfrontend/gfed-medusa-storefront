# Medusa Storefront Products

Products microfrontend for the Medusa storefront. Handles product-related routes including product details, collections, categories, and the store catalog.

## Storefront Development Server Setup Instructions

1. Set up the environment variables. Copy the `.env.template` to a local `.env` and update the values as needed.

```bash
cp .env.template .env
```

2. Start the development server:

```bash
pnpm run dev
```

The app will run on http://localhost:3001
